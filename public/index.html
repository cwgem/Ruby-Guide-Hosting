Introduction

This is a listing of the documentation for Ruby 1.9.3 preview 1. It's meant to be more up to date than what I've seen on the official ruby docs site. This currently runs on a free Heroku instance so the stability may vary.

Layout

The documentation is organized as follows:

  • Core Documentation - This is darkfish (rdoc) documentation for the core c based classes of Ruby.
  • C Extensions Documentation - This is darkfish (rdoc) documentation for the c based Ruby extensions. These were broken out so as to prevent confusion for implementations that do not support c extensions.
  • Standard Library Darkfish Docmentation (rdoc) - This is darkfish (rdoc) documentation for the Ruby based standard library.
  • Standard Library Yard Docmentation - This is yard (yardoc) documentation for the Ruby based standard library.
